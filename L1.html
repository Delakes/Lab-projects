Задание: Используя фреймовую структуру создать многостраничный web-сайт для отчетов о выполнении лабораторных работ. Залить полученные документы на сервер.

В качестве локального сервера было выбрано расширение для среды разработки Microsoft Visual Studio Code под названием "Live Server". Данное расширение создаёт локльный сервер на компьютере пользователя и, помимо прочего, позволяет динамически отслеживать изменения на странице.

Изображение 1. Отображение локального адреса страницы.
